# Uber E2E Frontend — Bloques 1 al 6

Avance acumulado del frontend React + TypeScript.

## Incluye

- Bloque 1: base del frontend, rutas y UI estilo Uber.
- Bloque 2: login y registro funcionales.
- Bloque 3: JWT en localStorage, Authorization Bearer y rutas protegidas por rol.
- Bloque 4: dashboard pasajero con GET /trips.
- Bloque 5: solicitar viaje con conductores disponibles y POST /trips.
- Bloque 6: detalle de viaje pasajero con GET /trips/:id, polling y formulario de calificación con POST /trips/:id/rate.

## Uso

Conserva tu archivo `.env` local y ejecuta:

```bash
npm install
npm run dev
```

Backend esperado:

```env
VITE_API_BASE_URL=http://localhost:8080
```
